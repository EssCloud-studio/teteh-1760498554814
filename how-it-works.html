Proses 4 Langkah Mudah

ESS Generator mempermudah proses pembuatan addon Minecraft menjadi 4 langkah sederhana

1

Konfigurasi Addon

Masukkan detail addon seperti nama, deskripsi, dan ikon. Atur versi @minecraft/server yang ingin digunakan.

2

Buat Command

Tambahkan custom command dengan menentukan nama, deskripsi, permission, dan aksi yang dijalankan.

3

Review & Atur

Lihat daftar command yang telah dibuat dan atur sesuai kebutuhan. Hapus atau edit jika diperlukan.

4

Generate & Download

Generate addon dalam format .mcaddon dan download file yang siap digunakan di Minecraft.

Teknologi di Balik Generator

JavaScript & Minecraft API

Generator menggunakan @minecraft/server API untuk membuat addon yang kompatibel dengan Minecraft Bedrock Edition.

Format .mcaddon

File yang dihasilkan menggunakan format .mcaddon standar yang langsung dapat diimpor ke Minecraft.

Client-Side Processing

Semua proses dilakukan di browser Anda. Tidak ada data yang dikirim ke server, menjamin privasi dan keamanan.
