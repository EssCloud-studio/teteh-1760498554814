Langkah 1: Detail Addon

Mulai dengan mengisi detail addon Anda. Beri nama yang unik, deskripsi yang jelas, dan pilih ikon untuk addon Anda. 

Nama Addon: "My Custom Commands"
Deskripsi: "Addon dengan command kustom untuk server"
Versi: @minecraft/server 2.3.0-beta

Langkah 2: Buat Command

Tambahkan command kustom dengan menentukan nama, deskripsi, permission level, dan aksi yang akan dijalankan. 

Command: "heal"
Deskripsi: "Memulihkan kesehatan pemain"
Permission: "Admin"
Aksi: "Jalankan Script Event"

Langkah 3: Generate & Download

Setelah semua command ditambahkan, generate addon dan download file .mcaddon yang siap digunakan. 

File Output: my_custom_commands.mcaddon
Format: Minecraft Addon Package
Siap diimpor ke Minecraft Bedrock

Script Event Command

Menjalankan script event yang dapat diproses oleh sistem script Minecraft. 

Command: /ess:heal
Aksi: scriptevent ess:heal

Kegunaan: Untuk command yang memerlukan logika kompleks atau integrasi dengan sistem lain.

Server Command

Menjalankan command Minecraft standar seperti /give, /tp, /effect, dll. 

Command: /ess:fly
Aksi: /ability @s mayfly true

Kegunaan: Untuk command sederhana yang menggunakan fungsi Minecraft yang sudah ada.

Message Command

Mengirim pesan teks kepada pemain yang menjalankan command. 

Command: /ess:info
Aksi: "Selamat datang di server kami!"

Kegunaan: Untuk memberikan informasi, petunjuk, atau pesan selamat datang.

ðŸ”§ Pemecahan Masalah Umum

Addon Tidak Berfungsi

Penyebab: Versi Minecraft yang tidak kompatibel atau kesalahan dalam konfigurasi.

Solusi:

  • Pastikan Minecraft Bedrock Edition versi 1.20.40+
  • Periksa kembali permission yang ditetapkan
  • Pastikan command name menggunakan huruf kecil tanpa spasi

File .mcaddon Tidak Dapat Diimpor

Penyebab: File corrupt atau struktur addon tidak valid.

Solusi:

  • Generate ulang addon
  • Pastikan semua field terisi dengan benar
  • Coba dengan command yang lebih sederhana terlebih dahulu

Command Tidak Terdaftar

Penyebab: Addon tidak aktif atau conflict dengan addon lain.

Solusi:

  • Pastikan addon aktif di Global Resources
  • Restart Minecraft setelah mengimpor addon
  • Periksa apakah ada addon lain dengan command serupa
